# Dino Game

Projeto para implementar o famoso Game do Dinossauro (quando estamos sem internet). <br />
A ideia é exercitar a parte de animações com HTML + CSS + JS. <br />
Hospedado no Github Pages.

<br />

### **Tecnologias utilizadas**
----
- HTML
- CSS
- JS
